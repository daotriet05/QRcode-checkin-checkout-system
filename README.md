# How to use
## Client
- to host Frontend, copy scripts below to command prompt:
- `cd client`
- `npm install`
- `npm run server`

## Server
- to host Backend, copy scripts below to command prompt:
- `cd server`
- `conda create -n env_name python==3.12`
- `conda activate env_name`
- `pip install -r requirements.txt`
- `python app.py`
